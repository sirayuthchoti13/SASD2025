# 1) Utility Class  
## *System.Math*

### Documentation
https://learn.microsoft.com/en-us/dotnet/api/system.math?view=net-10.0  

### Example

```csharp
double result = Math.Sqrt(25);
Console.WriteLine(result); // 5
```

### *Analysis*

* เป็น `static class`
* มีเฉพาะ `static methods`
* ไม่สามารถสร้าง instance ได้
* ไม่มี state ภายใน object

### *Conclusion*

*System.Math* เป็นคลาสสำหรับฟังก์ชันคำนวณทั่วไป เช่น  
* `Sqrt()`  
* `Pow()`  
* `Abs()`  

จัดอยู่ในกลุ่ม **Utility Class**

---

# 2) Mutable Class  
## *System.Text.StringBuilder*

### Documentation
https://learn.microsoft.com/en-us/dotnet/api/system.text.stringbuilder?view=net-10.0  

### Example

```csharp
using System.Text;

StringBuilder sb = new StringBuilder("Hello");
sb.Append(" World");

Console.WriteLine(sb.ToString()); // Hello World
```

### *Analysis*

* สามารถเปลี่ยนแปลงค่าได้หลังจากสร้าง object แล้ว
* เมธอด เช่น  
  * `Append()`  
  * `Insert()`  
  * `Remove()`  
  แก้ไข object เดิม
* มี internal state ที่เปลี่ยนแปลงได้

### *Conclusion*

*StringBuilder* จัดอยู่ในกลุ่ม **Mutable Class**

---

# 3) Immutable Class  
## *System.String*

### Documentation
https://learn.microsoft.com/en-us/dotnet/api/system.string?view=net-10.0  

### Example

```csharp
string s = "hello";
string t = s.Replace("h", "H");

Console.WriteLine(s); // hello
Console.WriteLine(t); // Hello
```

### *Analysis*

* ค่าไม่สามารถเปลี่ยนแปลงได้หลังจากสร้าง
* เมธอด เช่น  
  * `Replace()`  
  * `ToUpper()`  
  * `Trim()`  
  จะคืนค่าเป็น object ใหม่

### *Conclusion*

*System.String* จัดอยู่ในกลุ่ม **Immutable Class**

# Comparison

| Type | Example | Change State? |
|------|----------|--------------|
| Utility | `System.Math` | ❌ |
| Mutable | `StringBuilder` | ✅ |
| Immutable | `String` | ❌ |

---

# Summary

* **Utility Class** → มีแต่ static methods  
* **Mutable Class** → เปลี่ยนค่าได้  
* **Immutable Class** → เปลี่ยนค่าไม่ได้ ต้องสร้างใหม่
